Front-End Actividades EVG 🚀

Aplicación que se encarga de mostrar las vistas de principalmente los momentos y las actividades del colegio.

Instalar aplicación:



  • Descargar el proyecto

  • Ejecutar el comando npm install -g

  • Instalar angular npm install -g @angular/cli@13.3.3



Ejecutar proyecto: ng serve (acceder a través de localhost:4200)





Compilar el proyecto ng build (se guardará en la carpeta /dist)



Solucionar problemas con la instalación

Si has tenido algún problema con la instalación te recomendamos hacer lo siguiente:


  1. Desistalar Angular npm uninstall -g @angular/cli

  2. Borrar cache npm cache clean

    1. También se puede usar el siguiente (si estás usando npm > 5) npm cache verify

  3. Volver a instalar Angular (siguiendo el punto de instalación nuevamente)
